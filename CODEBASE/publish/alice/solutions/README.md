## 一、Alice Solutions 

欢迎你来开发自己的 solutions，`fork` 我们，给我们 `pull request`；或者直接发邮件到 [admin@aliceui.com](mailto:admin@aliceui.com)。solutions 可以是：

- 浏览器 bug 兼容问题的解决方案
- HTML5/CSS3 的跨浏览器解决方案

## 二、Solutions 支持

目前 Solutions 支持原生 CSS 和 Less CSS。分为是目录下的：

 - SOLUTION-NAME.css
 - SOLUTION-NAME.less
